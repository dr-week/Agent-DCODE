## 📥 Installation Steps

### Step 1: Open VS Code
Launch Visual Studio Code

### Step 2: Open Extensions Panel
- Press `Ctrl+Shift+X` (Windows/Linux)
- Or click the Extensions icon in the sidebar

### Step 3: Install from VSIX
- Click the "..." menu in the Extensions panel
- Select **"Install from VSIX..."**
- Navigate to: `./vsc-agent/dcode/agent-dcode-0.1.0.vsix`
- Click **Open**

### Step 4: Configure Backend
1. Open VS Code Settings (`Ctrl+,`)
2. Search for: `dcode.backendURL`
3. Set to: `http://localhost:5000`
4. Restart VS Code

---

## ✨ You're Ready!

### Use it:
- **Via Command:** Ctrl+Shift+P → "Send to DCODE AI"
- **Via Sidebar:** Click DCODE icon → Chat

---

## 🎯 Quick Test
1. Select some code in an editor
2. Press Ctrl+Shift+P
3. Type "Send to DCODE AI"
4. Get instant AI response!

---

**Troubleshooting:**
- ❌ "Module not found" → Reinstall from VSIX
- ❌ "No API response" → Check API key in settings
- ❌ Still not working → Press F5 in extension folder to debug
