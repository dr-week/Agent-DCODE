## 📥 Quick Install

### Option 1: Direct Installation (EASIEST)

**Step 1:** Download the .vsix file
```
agent-dcode-0.1.0.vsix (10 KB)
Location: Project root or vsc-agent/dcode/
```

**Step 2:** Open VS Code → Extensions (Ctrl+Shift+X)

**Step 3:** Click the "..." menu → "Install from VSIX"

**Step 4:** Select `agent-dcode-0.1.0.vsix`

**Step 5:** Configure API Key
- Settings (Ctrl+,) → Search "dcode.apiKey"
- Paste your OpenAI API key

**Done!** 🎉

## 🔍 Troubleshooting

### "Extension not found"
- Make sure VS Code version 1.110+ (check Help → About)
- Try uninstalling and reinstalling

### "API key error"
- Copy full key (including "sk-" prefix)
- Paste in Settings → dcode.apiKey
- Reload VS Code

### "No response from AI"
- Check internet connection
- Verify API key is valid
- Check API quota on OpenAI dashboard
